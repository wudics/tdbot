# 深色模式

## 目标

支持浅色/深色主题切换，默认浅色，设置弹窗中切换，持久化到配置，保存后即时生效。

## 实现原理

Tailwind CSS 的 `darkMode: ['class']` 模式通过给 `<html>` 添加 `dark` 类来触发所有 `dark:*` 样式和 CSS 变量覆盖。

## CSS 变量

### globals.css

在 `:root` 下定义浅色变量，`.dark` 下定义深色变量：

```css
:root {
  --background: 0 0% 100%;
  --foreground: 240 10% 3.9%;
  --primary: 240 5.9% 10%;
  --primary-foreground: 0 0% 98%;
  --muted: 240 4.8% 95.9%;
  --muted-foreground: 240 3.8% 46.1%;
  --border: 240 5.9% 90%;
  --input: 240 5.9% 90%;
  --ring: 240 5.9% 10%;
}

.dark {
  --background: 240 10% 3.9%;
  --foreground: 0 0% 98%;
  --primary: 0 0% 98%;
  --primary-foreground: 240 5.9% 10%;
  --muted: 240 3.7% 15.9%;
  --muted-foreground: 240 5% 64.9%;
  --border: 240 3.7% 15.9%;
  --input: 240 3.7% 15.9%;
  --ring: 240 4.9% 83.9%;
}
```

所有 Shadcn/ui 组件均使用这些 CSS 变量，切换 `dark` class 后自动适配。

## 数据流

```
SettingsDialog 切换 theme
  → config.theme 保存到 config.json
  → App.tsx 调用 applyTheme()
  → document.documentElement.classList.toggle('dark', theme === 'dark')
  → CSS 变量切换 → 整个应用深色
```

启动时 `App.tsx` 读取 `config.theme`，调用 `applyTheme()`。

## config.json 新增字段

```json
{
  "providers": [...],
  "activeProvider": "deepseek",
  "activeModel": "deepseek-v4-flash",
  "activeAgent": "general",
  "theme": "light",
  "webSearch": false
}
```
