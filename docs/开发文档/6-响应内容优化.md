# 响应内容优化

## 目标

优化 AI 回复内容的展示体验，包括代码块语法高亮、复制按钮、加载动画、思考过程折叠等。

## 1. 代码块语法高亮 + 复制按钮

### 技术选型

| 项目 | 选择 | 理由 |
|------|------|------|
| 高亮引擎 | **highlight.js** | 轻量，支持 197 种语言，无需额外配置 |
| 主题 | `github` | 与 Shadcn/ui 风格一致，浅色干净 |
| CSS 引入 | `globals.css` 中 `@import` | 零配置 |

### 安装

```bash
npm install highlight.js
```

### MessageItem 渲染变更

改前：整段文本 `whitespace-pre-wrap` 直接输出

改后：先 `parseCodeBlocks` 分割为 segments，逐段渲染：

```
text segment  → <span className="whitespace-pre-wrap">{content}</span>
code segment  → <CodeBlock language code />
```

## 2. 加载动画

AI 回复时正文区域显示动态 `...`：

```css
@keyframes blink {
  0%, 20% { opacity: 0; }
  50% { opacity: 1; }
  100% { opacity: 0; }
}
```

三个 `<span>` 分别以 `0s`、`0.2s`、`0.4s` 延迟循环动画。

## 3. 思考过程折叠

- 流式输出中：自动展开显示思考内容
- 输出完成后：自动折叠，显示 `▶ 思考过程` 按钮
- 点击切换展开/折叠
- 内容最多 10 行（`max-h-[15em] overflow-y-auto`），超出显示滚动条

## 5. 消息时间戳

每条消息右下角显示格式化时间：

### 格式化规则

```
今天消息       → 14:30
昨天消息       → 昨天 14:30
今年内的消息   → 6/4 14:30
更早的消息     → 2025/6/4 14:30
```

### 实现

`src/renderer/src/lib/formatTime.ts`：

```typescript
export function formatTime(isoString: string): string {
  const date = new Date(isoString)
  const now = new Date()
  const today = new Date(now.getFullYear(), now.getMonth(), now.getDate())
  const target = new Date(date.getFullYear(), date.getMonth(), date.getDate())
  const diffDays = Math.floor((today.getTime() - target.getTime()) / 86400000)

  const time = date.toLocaleTimeString('zh-CN', { hour: '2-digit', minute: '2-digit' })

  if (diffDays === 0) return time
  if (diffDays === 1) return `昨天 ${time}`
  if (date.getFullYear() === now.getFullYear()) {
    return `${date.getMonth() + 1}/${date.getDate()} ${time}`
  }
  return `${date.getFullYear()}/${date.getMonth() + 1}/${date.getDate()} ${time}`
}
```

### 数据流

1. `useChat.sendMessage()` 创建消息时设置 `createdAt = new Date().toISOString()`
2. 流式更新保持 `createdAt` 不变
3. `saveToStorage()` 将 `createdAt` 持久化到 `sessions.json`
4. `loadMessages()` 从存储加载时携带 `createdAt`
5. `MessageItem` 调用 `formatTime()` 渲染
