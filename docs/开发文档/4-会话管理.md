# 会话管理

## 目标

实现会话侧边栏，支持新建、切换、删除、重命名对话，消息本地持久化，以及响应式抽屉式侧边栏。

## 存储方案

本地 JSON 文件 `~/.config/tdbot/sessions.json`：

```json
{
  "sessions": [
    {
      "id": "ses_xxx",
      "title": "帮我写一个Python脚本",
      "createdAt": "2026-06-04T12:00:00Z"
    }
  ],
  "messages": {
    "ses_xxx": [
      {
        "id": "1",
        "role": "user",
        "text": "帮我写一个Python脚本",
        "reasoning": null,
        "createdAt": "2026-06-04T12:00:01Z"
      },
      {
        "id": "2",
        "role": "assistant",
        "text": "好的...",
        "reasoning": "用户需要...",
        "createdAt": "2026-06-04T12:00:10Z"
      }
    ]
  }
}
```

### sessionStore.ts 设计

```typescript
// 文件路径: app.getPath('userData')/sessions.json
loadAll()               // 读取完整 JSON
saveSessionList(list)   // 只写 sessions[] 段
saveMessages(id, msgs)  // 只写 messages[id] 段
deleteSession(id)       // 删除会话及其消息（本地 + opencode server）
```

删除和重命名**同步**到 opencode server：

```typescript
// 删除 — 同时调用 server API
await opencodeClient.session.delete({ sessionID: id })
deleteStoredSession(id)

// 重命名 — 同时调用 server API  
await opencodeClient.session.update({ sessionID: id, title })
saveSessionList(sessions)
```

## IPC 协议

| 方向 | 方法 | 参数 | 返回 | 说明 |
|------|------|------|------|------|
| → | `list-sessions` | — | `SessionMeta[]` | 返回本地存储的会话列表 |
| → | `create-session` | — | `SessionMeta` | 先在 opencode 创建，再存本地 |
| → | `switch-session` | `id` | `StoredMessage[]` | 从本地加载消息 |
| → | `delete-session` | `id` | `void` | 删除本地 + opencode |
| → | `update-session-title` | `id, title` | `void` | 修改标题 |
| → | `save-messages` | `sessionId, msgs` | `void` | 流结束时自动调用 |
| → | `abort-message` | — | `void` | 终止当前正在生成的响应 |
| → | `send-message` | `text, sessionId` | `{ success }` | 不再内部自动创建 session |

## 标题生成机制

1. 创建时标题为"新建对话"
2. 流式回复完成后，取用户的首条消息前 30 字作为标题
3. 用户可双击侧边栏标题直接编辑

## 加载动画

AI 回复时，正文区域显示动态 `...` 通过 CSS keyframe 实现：

```css
@keyframes blink {
  0%, 20% { opacity: 0; }
  50% { opacity: 1; }
  100% { opacity: 0; }
}
```

三个 `<span>` 分别以 `0s`、`0.2s`、`0.4s` 延迟循环动画。

## 思考过程折叠

- 流式输出中：自动展开显示
- 输出完成后：自动折叠，显示 `▶ 思考过程`
- 点击切换展开/折叠
- 内容最多 10 行（`max-h-[15em] overflow-y-auto`），超出显示滚动条

## 终止输出

发送消息后输入框禁用，发送按钮变为红色停止按钮（■）。
点击停止 → IPC `abort-message` → `client.session.abort({ sessionID })` → opencode 停止生成。

## 数据流

```
发送消息:
  InputBar → useChat.sendMessage(text)
    → onStream 更新 messages 状态 + 同步 messagesRef
    → onEnd → saveToStorage() + onAutoTitle()
    → 异常 → onError()

切换会话:
  SessionSidebar → useSessions.switchSession(id)
    → 先 await switchSession 获取消息 → 再 setActiveId
    → ChatView 因 key={activeId} 重新挂载
    → useChat.loadMessages(initialMessages) + setSessionId(id)

终止生成:
  InputBar.按钮 → useChat.abort()
    → IPC abort-message → session.abort({ sessionID })
    → server 触发 session.error → stream-end → UI 恢复
```
