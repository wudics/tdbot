# 目的
我想要基于opencode的sdk开发一个桌面版的智能体，可以通过简单的聊天窗口与用户交互，完成用户的复杂任务。

# 架构
- 跨平台：可以在linux、windows、macos上运行
- 前端：使用electron，实现界面展示、用户交互和系统配置等功能
- 后端：使用opencode的sdk和api实现智能体的逻辑
- 知识库：weknora是单独的一个模块，知识问答时可指定知识库问答，单独创建weknora页面通过api管理知识库
- 智能体：使用opencode作为智能体，在当前项目下安装opencode，以便electron打包（opencode是二进制文件）

# 功能
- 支持用户上传word、execl、pdf等文本文件，并基于上传文本文件进行问答
- 支持用户上传图片、语音、视频等，并基于上传多媒体文件进行问答
- 支持基于weknora知识库问答
- 支持互联网搜索开关，默认关闭（如果opencode sdk不支持，请提示）
- 支持会话管理
- 支持多轮对话

# npm国内镜像源
如果遇到npm下载安装依赖包时，请使用国内镜像源：
npm config set registry https://registry.npmmirror.com/

# electron国内镜像源
在打包时，如果需要下载安装electron的话，请使用以下环境变量，配置国内镜像源

- ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
- ELECTRON_BUILDER_BINARIES_MIRROR=https://npmmirror.com/mirrors/electron-builder-binaries/
